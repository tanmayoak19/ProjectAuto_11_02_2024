
**********************************************************************************************************************************
This is the Page Factory framework in Java-selenium using TestNg and Maven
**********************************************************************************************************************************

-   It consist of pom.xml file which has all the dependencies that are required for project

1.  This Framework consist of Driver setup file for setting up the driver.
2.  It has a base page which include all the basic methods like (i.e selenium waits,javascript executor methods,driver methods)
2.  It has Locator Pages for the particular pages.
3.  It has Method Pages that includes all the important methods regarding that page.
4.  We have Script package which contains testcases related to particular section according to modules.

**********************************************************************************************************************************


**********************************************************************************************************************************
How to Run The TC:-
**********************************************************************************************************************************

1.  There are TestNg file which include testcases according to folders.
2.  So we can run the Test-Ng files and get the results as per need.
